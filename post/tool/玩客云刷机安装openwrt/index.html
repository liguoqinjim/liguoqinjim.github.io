玩客云刷机安装openwrt

2023-07-18
 | 玩客云 tool
 | reads
Table of Contents

玩客云刷机，安装openwrt，作为软路由使用。

准备

  1. 公对公USB线一根
  2. SD卡（U盘也可以，但是安装openwrt的时候U盘的方式可能会有问题，推荐SD卡）
  3. 镊子（或者任何可以用来短接的东西）
  4. 吹风机（加热面板，方便拆机）

软件：

百度网盘：链接: https://pan.baidu.com/s/1i-bW975Pdwg2u0QonMFgwg?pwd=0718 提取码: 0718 。

拆机

  1. 使用吹风机对面板加热
  2. 撬开面板
  3. 拧开螺丝即可

刷机

  1. 安装Amlogic.USB.Burning.Tool.v2.1.6.8.exe

  2. 打开软件，载入镜像 “s805_flash_snail.dmg”，然后点击开始。

  3. 连接公对公 USB 线。

    1. 注意端口，需要使用玩客云上靠近 HDMI 一侧的 USB 口。
    2. 电脑端最好使用和 USB 线一致的端口，如果是 USB 2.0 的线就使用 USB 2.0 的口，避免刷机报错。

  4. 进行玩客云的短接。

    新老板子的短接方法不一样，我使用的是老版的。

  5. 插上电源。

  6. 顺利的话，就开始刷机了，软件显示成功之后，直接断电并关闭软件即可。

  7. 刷机成功！这一步可以使我们能够从 U 盘或 SD 卡启动固件。

可能遇到的问题：

  1. 刷机软件没有反应

    确认是否成功进行了短接操作。

  2. 刷机软件报错：usb命令不对等等

    • 第一种方式是重新按照步骤再试几次。

    • 如果仍然不行，可以尝试按下板子上的四个弹片，使其与板子接触，然后再进行刷机。

我就是在按下了四个弹片并持续进行短接的情况下，经过了二十多次刷机报错，最终成功了。大家加油！

安装openwrt

  1. 安装balenaEtcher
  2. U盘刷入openwrt固件，我在共享文件里面分享了两个openwrt固件。
  3. 连接 U 盘并开机（连接到靠近网卡那一侧的端口）。
  4. LED 将会显示蓝灯，然后闪烁红色，再变成蓝色，表示成功启动 OpenWrt。

可能遇到的问题：

  1. U盘连接没有反应

    • 尝试两个 USB 接口。

    • 尝试使用 SD 卡。
