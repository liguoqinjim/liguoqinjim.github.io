申请免费域名eu.org

2023-04-04
 | 工具 tool
 | reads

申请eu.org的免费域名，再配合cloudfare申请无限邮箱。

申请域名

登录https://nic.eu.org/ 注册账号。

  1. nameserver可以先用腾讯的，之后可以换。
  2. 提交了申请之后，等待的时候不确定。有的很快，有的很慢，我等了1个月。可以关注下邮箱的垃圾箱，我的通过邮件是默认分类垃圾箱了。
  3. ❗申请的时候：name servers选择 server names才可以，不要用server names+replies on SOA + replies on NS。
  4. 提交页面出现Done就说明提交成功了，如果提示有errors的话则提交无效。

域名转入cloudfare

域名通过之后，可以把域名转入cloudfare来管理。

  1. 在cloudfare页面添加域名
  2. 把域名的nameserver换成cf的
  3. 等待cf确认就行了

cf无限邮箱

选择cf中的Email：

根据提示自动添加ns record，添加完成后，配置邮箱。

这样配置完成后，发送到Custom address的邮件就可以在Destination邮箱中收到了。
