设置markdown里面的图片大小

 | markdown
 |

在markdown里面添加图片的时候，图片显示出来可能会比我们需要的大很多。 这时候我们会需要设置图片的大小。

1.markdown语法添加图片

![webhooks](/images/IFTTT/snipaste20180702_184145.png)

但是这样的添加方式，我们不能设置图片的大小。要设置大小需要使用HTML标签

2.通过HTML标签添加图片并设置图片大小

<img src="/images/IFTTT/snipaste20180702_184145.png" width="200" alt="webhooks">

width可以用来设置图片的大小

3.有的需要根据使用的主题的规则来设置

有的主题里面使用<img>标签没用，这个可能是主题有自己的图片规则，例如hugo-tranquilpeak-theme，这个时候具体查看主题的文档就可以了。
