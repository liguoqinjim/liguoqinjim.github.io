在windows下手动安装mysql

下载

下载地址：https://dev.mysql.com/downloads/mysql/5.7.html，下载zip格式
解压之后的文件目录：

安装

1.创建配置文件my.ini

[mysqld]

# mysql运行的端口
port=3306
# mysql解压之后的路径
basedir=D:/workspace/mysql-5.7.28-winx64
# mysql存放数据的路径
datadir=D:/workspace/3306/data

2.安装mysql服务

命令：mysqld install 3306 --defaults-file="D:\workspace\mysql-5.7.28-winx64\bin\my.ini"，3306是服务名称，--defaults-file指定配置文件的路径
安装完成之后，可以在系统服务里面找到，如：

3.初始化数据

手动从zip包安装的时候，需要手动初始化数据
命令：mysqld --defaults-file="D:\workspace\mysql-5.7.28-winx64\bin\my.ini" --initialize --console
--intialize可以换成--initialize-insecure，两者区别是--intialize会给root用户生成一个随机密码，会输出在命令行中。--intialize-insecure会给root用户一个空密码。

启动服务

命令：net start 3306，3306就是服务名

删除服务

命令：mysqld remove 3306
